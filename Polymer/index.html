UI News Reader - Polymer Web Component

Info

"UI News Reader" is web component that gets news from rss urls and shows in page easily. It requires Polymer Framework and url that must response data in RSS JSON format.

It's created by İlker Güller.

Attributes

  • url Required (string) : JSONP Url where component get data response from
  • requireImages Optional (boolean) : Set true to get news that doesnt has image
  • changeUrlPath Optional (boolean) : Set true to change page url with encoded news title automatically
  • changeTitle Optional (boolean) : Set true to change page title with news title automatically

SASS Attributes

  • $componentWidth : Component Width
  • $componentHeight : Component Height
  • $colorWhite : White Hex Color
  • $colorWhiteSmoke : White Smoke Hex Color
  • $colorGray : Gray Hex Color
  • $colorBlackText : Black Hex Color for Texts
  • $colorOrangeHover : Orange Hex Color for hover effects
  • $fontUbuntu : Ubuntu Font Family
  • $sizeBig : Text Size
  • $sizeNormal : Text Size
  • $sizeSmall : Text Size

Example

Basic
<ui-newsreader url="http://pipes.yahoo.com/pipes/pipe.run?_id=DqsF_ZG72xGLbes9l7okhQ&_render=json&_callback="></ui-newsreader>
With Attributes
<ui-newsreader url="http://pipes.yahoo.com/pipes/pipe.run?_id=DqsF_ZG72xGLbes9l7okhQ&_render=json&_callback=" requireImages="false" changeUrlPath="false" changeTitle="true" ></ui-newsreader>
